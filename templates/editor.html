🛠️ 다이어리 빌딩 가이드

  1. 페이지 추가: 좌측 + 추가를 눌러 필요한 플래너(먼슬리, 데일리 등)를 AI로 각각 개별 생성합니다.
  2. 페이지 복제: 일기나 할일 목록처럼 매일 쓰는 양식은 복제 버튼(📋)을 눌러 날짜별로 빠르게 여러 장 생성하세요.
  3. 하이퍼링크 생성: 상단의 링크 연결 모드를 켜고, 캘린더나 특정 텍스트를 클릭한 뒤 대상 페이지를 선택하면 클릭 가능한 PDF 링크가 자동으로 바인딩됩니다.
  4. 링크 작동 확인: 링크 연결 모드를 끄고 페이지의 링크 영역을 누르면 실제 연결된 페이지로 즉시 점프합니다.
